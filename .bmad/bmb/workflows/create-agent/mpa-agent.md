# Create BMAD Agent Workflow

## Overview
This workflow guides the creation of a new BMAD agent following best practices and compliance standards.

## Steps

### Step 1: Agent Definition
- Gather agent requirements from user
- Define agent purpose and scope
- Identify target platform and capabilities needed

### Step 2: Agent Structure Creation
- Create agent YAML file with proper structure
- Define activation sequence
- Configure persona and communication style
- Set up menu options and handlers

### Step 3: Compliance Check
- Validate agent structure against BMAD standards
- Ensure proper YAML formatting
- Check for required sections and fields

### Step 4: Testing
- Test agent activation
- Verify menu functionality
- Validate workflow handlers

## Required Files
- Agent YAML file in `.bmad/bmb/agents/` directory
- Optional: Custom workflow files in `.bmad/bmb/workflows/` directory

## Output
- Fully functional BMAD agent
- Compliance validation report
- Testing results
