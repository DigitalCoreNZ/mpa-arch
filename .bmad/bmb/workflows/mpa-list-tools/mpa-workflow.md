# List Available Tools Workflow

## Overview
This workflow displays all available tools and their capabilities for meta prompt creation.

## Steps

### Step 1: Tool Enumeration
- Load tool capabilities from agent configuration
- Categorize tools by function type
- Prepare detailed descriptions

### Step 2: Browser Access Tools
- Web browsing capabilities
- Page content extraction
- Form interaction
- JavaScript execution

### Step 3: Internet Access Tools
- HTTP requests
- API calls
- Data fetching
- Authentication handling

### Step 4: MCP Protocol Tools
- MCP server connections
- Context management
- Protocol messaging
- Resource sharing

### Step 5: Task Management Tools
- Task creation
- Subtask management
- Progress tracking
- Dependency handling

### Step 6: OS Command Tools
- Shell command execution
- System operations
- Process management
- Environment variables

### Step 7: File Operation Tools
- File creation
- File reading
- File updating
- File deletion
- Directory management

### Step 8: Playground Tools
- Bash script testing
- Python script testing
- TypeScript program testing
- Environment validation

## Output
- Comprehensive tool list with descriptions
- Usage examples for each tool
- Integration guidelines
