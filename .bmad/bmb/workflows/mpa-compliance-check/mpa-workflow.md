# Agent Compliance Check Workflow

## Overview
This workflow validates the MetaPromptAgent against BMAD standards and best practices.

## Steps

### Step 1: Structure Validation
- Verify YAML file format
- Check required sections presence
- Validate agent metadata
- Ensure proper indentation and syntax

### Step 2: Activation Sequence Check
- Verify activation steps completeness
- Check configuration loading
- Validate prerequisite checks
- Ensure proper error handling

### Step 3: Persona Validation
- Check persona definition completeness
- Verify communication style consistency
- Validate principles alignment
- Ensure role clarity

### Step 4: Capabilities Verification
- Validate tool capability definitions
- Check function completeness
- Verify tool descriptions
- Ensure proper categorization

### Step 5: Menu System Check
- Verify menu item definitions
- Check handler references
- Validate command triggers
- Ensure proper formatting

### Step 6: Workflow Integration
- Verify workflow file existence
- Check handler path validity
- Validate workflow content
- Ensure proper integration

### Step 7: Compliance Report
- Generate compliance status
- List any violations
- Provide recommendations
- Create remediation plan

## Required Files
- Agent YAML file
- All referenced workflow files
- BMAD compliance standards

## Output
- Compliance validation report
- Violation details (if any)
- Remediation recommendations
- Final compliance status
